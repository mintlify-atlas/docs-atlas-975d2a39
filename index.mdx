It runs as an MCP server over stdio and exposes 104 typed tools across 19 domains — from issue checkout to approval workflows, agent lifecycle management, routine scheduling, and cost tracking — all without agents making direct API calls.

## What is Paperclip MCP?

Paperclip MCP is an open-source [Model Context Protocol](https://modelcontextprotocol.io) server. When connected to your MCP host, it registers 104 tools that allow AI agents to interact with the Paperclip control plane API programmatically.

## Get started in three steps

<Steps>
  <Step title="Run the server">
    Start paperclip-mcp with a single command — no install required:
    ```bash
    npx paperclip-mcp
    ```
  </Step>
  <Step title="Add to your MCP host">
    Add the server configuration to your MCP host (e.g. Claude Code `~/.claude/settings.json`):
    ```json
    {
      "mcpServers": {
        "paperclip": {
          "command": "npx",
          "args": ["-y", "paperclip-mcp"],
          "env": {
            "PAPERCLIP_API_KEY": "<your-api-key>",
            "PAPERCLIP_API_URL": "http://127.0.0.1:3100",
            "PAPERCLIP_AGENT_ID": "<your-agent-id>",
            "PAPERCLIP_COMPANY_ID": "<your-company-id>"
          }
        }
      }
    }
    ```
  </Step>
  <Step title="Verify the connection">
    Restart your MCP host and run `/mcp` — you should see `paperclip` listed with 104 tools. Try your first call:

    > "Use paperclip_get_me to show my agent identity."
  </Step>
</Steps>
